# Practica 17 - Repaso Componentes II

Proyecto: repaso2  
Aplicacion: Registro de Libros Leidos / Catalogo de Libros

## Requerimientos cubiertos

- SplashScreen de bienvenida durante 2 segundos.
- Pantalla principal con ImageBackground.
- Formulario con TextInput para titulo del libro, autor y genero.
- Boton creado con Pressable para agregar libro.
- Validacion de campos obligatorios.
- Espera simulada de 4 segundos usando ActivityIndicator.
- Registro de libros en FlatList.
- Limpieza de TextInput despues de guardar.
- Notificacion con Alert.

## Como ejecutar

```bash
npm install
npx expo start
```

Despues escanea el QR con Expo Go desde el telefono.
